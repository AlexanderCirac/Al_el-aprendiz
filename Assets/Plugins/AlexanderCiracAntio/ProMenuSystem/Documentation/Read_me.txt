Español------------------------------------

Para poder utilizar este sistema, solo tienes que  coger la script “SC_ProMenuCore”

English-----------------------------------------

Thank you very much for purchasing the package! I hope you find it very useful.
To be able to use this system, you only have to take the script "SC_ProMenuCore"
and drag it to any object on the stage, this will be added automatically inside
its scene an empty object with the name "ProMenuMannager", in which
they will automatically place all the scripts you will need.

-SC_ProMenuExplication
-SC_ProMenuUIMannager
-SC_ProMenuOptionsMannager
-SC_ProMenuLoadBarMannger

For more instructions to the pdf or on the website
